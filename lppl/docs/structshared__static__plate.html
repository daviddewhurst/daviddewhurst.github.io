Detailed Description

template<typename D, size_t N>
struct shared_static_plate< D, N >

Defines a generic shared pointer plated distribution.

  • Suppose that \( z \sim p(z|D) \). Then this struct defines the distribution \( z_1,...,z_N \sim \prod_{1\leq n \leq N} p(z_n|D)\).
Template Parameters
Dthe underlying distribution type
Nthe compile-time constant dimensionality of the plate
