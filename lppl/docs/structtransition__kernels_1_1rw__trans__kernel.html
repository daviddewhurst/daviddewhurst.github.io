Detailed Description

template<typename D, typename... Ts>
struct transition_kernels::rw_trans_kernel< D, Ts >

A generic double-valued random walk transition kernel.

This kernel assumes the distribution to be superceded \(p\) is in the location-scale family. Denote the random variable by \(z\). Given a user-defined scale \(\sigma\) and an unweighted cache \(D_{t-1}(z)\) of previous samples of \(z\), this distribution corresponds to \(z \sim p\left( E_{z \sim D_{t-1}(z)}[z], \sigma \right) \).
