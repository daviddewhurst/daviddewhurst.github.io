template<typename, typename O, typename... Ts>
class WeightedRecord< typename, O, Ts >

A collection of weighted records. This is the "traditional" empirical posterior distribution as implemented by many trace-based probabilistic programming languages. The memory use of this object scales as O(num_samples).
